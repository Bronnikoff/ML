Задача - реализовать линейную регрессию с помощью метода градиентного спуска с использованием регуляризации и побить заданный бейзлайн на приват-части.

Небольшой роадмеп в помощь:
зарегистрировать на сайте kaggle
присоединиться к соревнованию и изучить данные здесь:
train.csv - это данные, на которых стоит обучать модель
test.csv - это данные, на которых нужно предсказать
sample.csv - это как должны выглядеть предсказания вашей модели
написать модель градиентного спуска с регуляризатором, обучить модель на данных train.csv, пока в данном задании можно использовать только следующие данные для обучения, то есть мы работаем только с переменными типа int64:

['Id', 'MSSubClass', 'LotArea', 'OverallQual', 'OverallCond',
'YearBuilt', 'YearRemodAdd', 'BsmtFinSF1', 'BsmtFinSF2', 'BsmtUnfSF',
'TotalBsmtSF', '1stFlrSF', '2ndFlrSF', 'LowQualFinSF', 'GrLivArea',

'ScreenPorch', 'PoolArea', 'MiscVal', 'MoSold', 'YrSold', 'SalePrice']


прислать предсказание модели для данных test.csv, предварительно оценив качество модели на отложенной выборке. Пусть Вас не смущает заданная метрика соревнования RMSLE - оптимизируйте в ваших алгоритмах MSE, а качество на отложенной выборке проверяйте уже с помощью RMSLE, чтобы сравнивать качество на отложенной выборке и что у вас получается на лидерборде.

решение вашей задачи должно быть выполнено в Kernel (это обычный виртуальный ноутбук), посмотрите, как можно подгружать данные соревнований и как к ним обращаться на примерах других соревнований на Kaggle

получить скор для вашего ноутбука и расшарить его для всех преподавателей: nord786, aviator1337, izmaylov

обратите внимание, что вам нужно побить бейзлайн именно на тестовой ПРИВАТ части, которая вам не видна, то есть необходимо сделать решение, которое хорошо обобщает закономерности, а не подстраивается под текущий лидерборд. Используйте данные лидерборда как дополнительную отложенную выборку для измерения качества вашей модели.
